# WINDBOT Dataset Card / Thẻ Dữ liệu WINDBOT

## Dataset Description / Mô tả Dữ liệu

### Summary / Tóm tắt

WINDBOT is a bilingual (English-Vietnamese) wind turbine knowledge base designed for an AI chatbot serving the Qualcomm VR Lab project at EPU (Electric Power University). The dataset powers a RAG (Retrieval-Augmented Generation) system for answering questions about wind turbine technology, maintenance, operations, and safety.

WINDBOT là kho kiến thức song ngữ (Anh-Việt) về tua-bin gió, được thiết kế cho chatbot AI phục vụ dự án Qualcomm VR Lab tại Trường Đại học Điện lực (EPU). Bộ dữ liệu hỗ trợ hệ thống RAG để trả lời câu hỏi về công nghệ, bảo trì, vận hành và an toàn tua-bin gió.

### Supported Tasks / Nhiệm vụ Hỗ trợ

- Question answering about wind turbine technology / Trả lời câu hỏi về công nghệ tua-bin gió
- Technical term lookup and translation (EN↔VI) / Tra cứu và dịch thuật ngữ kỹ thuật
- Maintenance procedure guidance / Hướng dẫn quy trình bảo trì
- Safety protocol information / Thông tin quy trình an toàn
- Troubleshooting assistance / Hỗ trợ xử lý sự cố

### Languages / Ngôn ngữ

English (en), Vietnamese (vi)

## Data Processing Pipeline / Quy trình Xử lý

1. **Document Parsing / Phân tích tài liệu**: LlamaParse (PDF/DOCX/PPTX/XLSX) hoặc SimpleDirectoryReader (TXT/MD/CSV)
2. **Chunking / Phân đoạn**: SentenceSplitter (1024 tokens, 200 overlap)
3. **Embedding / Nhúng vector**: OpenAI text-embedding-3-small (1536 dimensions)
4. **Storage / Lưu trữ**: Supabase pgvector with cosine distance index

### Vector Store Configuration / Cấu hình Vector Store

- Collection: `wind_turbine_docs`
- Dimensions: 1536
- Distance metric: Cosine
- Chunk size: 1024 tokens
- Chunk overlap: 200 tokens

## Intended Use / Mục đích Sử dụng

This dataset is intended for:
- Training and evaluating the WINDBOT AI chatbot
- Supporting wind turbine education at EPU/TAF
- Powering the VR360 interactive learning experience

Bộ dữ liệu này dùng cho:
- Huấn luyện và đánh giá chatbot AI WINDBOT
- Hỗ trợ đào tạo về tua-bin gió tại EPU/TAF
- Cung cấp trải nghiệm học tập tương tác VR360

## Limitations / Hạn chế

- Dataset coverage depends on available technical documents from EPU/TAF / Độ bao phủ phụ thuộc vào tài liệu từ EPU/TAF
- Q&A corpus generated by LLM requires expert review / Bộ Q&A sinh bởi LLM cần chuyên gia review
- Vietnamese terminology may vary across regions / Thuật ngữ tiếng Việt có thể khác nhau giữa các vùng
- No real-time data integration / Không có tích hợp dữ liệu thời gian thực
