## 2. Quý vị cần chuẩn bị

### Người tham dự

Đề xuất cử **2-3 người** với các vai trò:

| Vai trò | Mô tả | Cần thiết? |
|---------|-------|------------|
| **Người quản lý** | Ra quyết định, ký biên bản | Bắt buộc |
| **Người vận hành** | Quản lý chatbot hàng ngày (thêm tài liệu, xem chat) | Bắt buộc |
| **Nhân viên kỹ thuật** | Xử lý sự cố, quản lý server (biết SSH, Linux cơ bản) | Khuyến nghị |

### Thiết bị

- [ ] Laptop có kết nối internet
- [ ] Trình duyệt Chrome hoặc Firefox (phiên bản mới nhất)
- [ ] Phòng họp có projector/TV và wifi ổn định

### Tài liệu mẫu

- [ ] Chuẩn bị 1 file tài liệu kỹ thuật tua-bin gió (PDF hoặc DOCX) để thực hành nạp vào hệ thống trong buổi đào tạo

### Thông tin cần xác nhận trước

Vui lòng phản hồi email kèm:

1. **Danh sách người tham dự** (họ tên, chức vụ, email)
2. **Thông tin phòng họp** (địa chỉ, có projector/wifi không?)
3. **Lựa chọn tài khoản OpenAI:**
   - Quý vị muốn **tạo tài khoản OpenAI riêng** (tự quản lý chi phí)?
   - Hay **nhận API key** từ nhóm phát triển?
4. **Lựa chọn nhận source code:**
   - Qua **GitHub** (cần email để mời vào repository)?
   - Qua **USB** (bản offline)?
   - Cả hai?

## 3. Sản phẩm bàn giao

Quý vị sẽ nhận:

| Hạng mục | Chi tiết |
|----------|----------|
| Source code | Frontend (Next.js) + Backend (FastAPI) — đầy đủ, chạy được |
| Knowledge base | Tài liệu kỹ thuật tua-bin gió đã nạp sẵn (2900+ chunks, 87 thuật ngữ) |
| Tài liệu hướng dẫn | Quản trị hệ thống, xử lý sự cố, phục hồi sự cố, migration |
| Access credentials | API keys, VPS access, database access |
| USB backup | Bản offline source code + data |
| Hỗ trợ 3 tháng | Sửa lỗi, hướng dẫn, hỗ trợ vận hành (theo thỏa thuận SLA) |

## 4. Chi phí vận hành dự kiến

Sau bàn giao, chi phí vận hành hàng tháng:

| Dịch vụ | Chi phí ước tính | Ghi chú |
|---------|-----------------|---------|
| OpenAI API | ~$5-20/tháng | Tùy lượng người dùng |
| Supabase (Database) | $0-25/tháng | Free tier hoặc Pro |
| VPS (Máy chủ) | Tùy provider | Chi phí hiện tại sẽ thông báo |
| Vercel (Website) | $0/tháng | Hosting miễn phí |
| **Tổng** | **~$5-45/tháng** | |

Chi tiết sẽ được trình bày trong buổi bàn giao.
